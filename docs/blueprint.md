# **App Name**: HeightWise

## Core Features:

- Height Generation: Generate a height within the range of 5'6" to 6' for typical names using an AI tool.
- Name Specific Logic: If the provided name is 'Kanish,' the tool should return a height between 3' and 4.5' and flag BMI as 'Too Obese.'
- BMI Calculation: Calculate BMI (Body Mass Index) based on the generated height and a default weight.
- Results Display: Display the generated height and BMI in an easily understandable format.
- Interactive UI: Create an interactive UI with input fields for name. It allows the AI tool to calculate the respective response.

## Style Guidelines:

- Primary color: Soft Blue (#A0D2EB) for a calm and trustworthy feel.
- Background color: Light Gray (#F0F4F8) for a clean, modern look.
- Accent color: Coral (#FF7F50) for interactive elements and highlighting.
- Body and headline font: 'Inter' sans-serif for a modern and readable style.
- Use simple, outlined icons to represent height and BMI.
- Design a clean layout with clear sections for input and results.
- Incorporate smooth transitions and subtle animations on user interactions.
